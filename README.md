# VAFPY Code
[![DOI](https://zenodo.org/badge/793990418.svg)](https://doi.org/10.5281/zenodo.17733769)


 This is a Auxiliary Field Quantum Monte Carlo (AFQMC) that is developed at the University of Vienna, Vienna Ab Initio Software Package (VASP).

 Code is written by Sajad Bazrafshan, Amir Taheridehkordi, Martin Schlipf, Zoran Sukurma, Moritz Humer, and Georg Kresse. 

 The Code can run with Numpy, Cupy, and Jax, which can work on CPU and GPU, and it works on single and double precision.
