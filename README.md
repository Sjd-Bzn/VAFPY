# VAFPY Code 

 This is a Auxiliary Field Quantum Monte Carlo (AFQMC) that developed at the University of Vienna, Vienna Ab Initio Software Package (VASP).

 Code is written by Sajad Bazrafshan, Amir Taheridehkordi, Martin Schlipf, Zoran Sukurma, Moritz Humer, Georg Kresse. 

 The Code can run with Numpy, Cupy, and Jax, which can works on CPU and GPU, and it can works on single and double precision.
